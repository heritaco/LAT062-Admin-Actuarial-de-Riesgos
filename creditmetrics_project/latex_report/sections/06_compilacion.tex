El proyecto LaTeX se encuentra en la carpeta \texttt{latex\_report/}. La compilaci\'on recomendada desde PowerShell es:

\begin{verbatim}
cd C:\Users\herie\Downloads\creditmetrics_project\mnt\data\creditmetrics_project
cd latex_report
pdflatex -interaction=nonstopmode -halt-on-error main.tex
pdflatex -interaction=nonstopmode -halt-on-error main.tex
\end{verbatim}

Tambi\'en se puede usar el script \texttt{build.ps1} incluido en esta carpeta.

Este documento referencia las im\'agenes en \texttt{../outputs/figuras/}. Por eso, antes de compilar conviene asegurarse de que el proyecto en Python ya haya sido ejecutado y haya generado las salidas visuales.

Archivos principales de salida:

\begin{itemize}
  \item \texttt{outputs/reporte\_creditmetrics.pdf}: reporte visual generado por Python;
  \item \texttt{outputs/resultados\_creditmetrics.xlsx}: libro Excel con tablas;
  \item \texttt{outputs/figuras/*.png}: tablas y gr\'aficas usadas por este documento;
  \item \texttt{latex\_report/main.pdf}: versi\'on LaTeX de explicaci\'on e interpretaci\'on.
\end{itemize}
