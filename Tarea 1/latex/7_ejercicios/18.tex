\subsection{FRM Exam 2008: 4-sigma daily event on a \$150M portfolio}

\textbf{Enunciado}

\emph{John Flag, the manager of a \$150 million distressed bond portfolio, conducts stress tests on the portfolio.
The portfolio’s annualized return is 12\%, with an annualized return volatility of 25\%.
If the portfolio would suffer a 4-sigma daily event, estimate the change in the value of this portfolio.}
\begin{enumerate}[label=\alph*.]
\item \hl{\$9.48 million}
\item \$23.70 million
\item \$37.50 million
\item \$150 million
\end{enumerate}

\subsubsection*{Solución}


\textbf{Convertir volatilidad anual a diaria}. Con \(252\) días hábiles/año:
\[
\sigma_d=\frac{0.25}{\sqrt{252}}\approx 0.01575=1.575\%.
\]

\textbf{Evento 4-sigma}\\
\[
|r|\approx 4\sigma_d\approx 4(0.01575)=0.0630=6.30\%.
\]

\textbf{Cambio de valor. }Si \(V=150\) millones:
\[
|\Delta V|\approx V|r|\approx 150(0.0630)=9.45\text{M}\approx 9.48\text{M}.
\]
